遷 於 銀湖 之北 。 中 更 一徙 。 以襲 其初 。 傳記 缺遺 。 莫詳 歲 月 。 嗟夫 。 昔穢國 盡燒 。 我淨 土弗毀 。 慧眼 之 覩 矣 。 今精 廬斯壞 。 我法侶 無歸 。 凡目 之取 矣 。 有禪 師德號 道融 。 本姓 樓 。 東陽義烏人 也 。 肅 宗皇帝 龍飛 朔方 。 大赦天下 。 改元 為 至德 。 每寺度 人 。 以蕃 王室 。 時潤州 刺史 兼 御史大夫 江南 東道 節度 處置 使 京兆 韋公陟 。 俾屬 城 大德 。 咸舉 所知 。 禪師 行業 精修 。 法門 之中 。 裒 然為 首 。 遂 正名 僧籍 。 而 闕 一字 於 福興 焉 。 初入 牛頭山 。 謁 第六 闕 一字 忠大師 。 遽 受密 印而為 正座 。 大師 三昧 之主 。 四支 之尊 。 攝心 無涯 。 定力 無 等 。 首施 錢 三十 萬 。 謂 禪師 曰 。 可闕 二字 招提 。 闕 一字 其安樂 。 禪師 跪 以 奉命 。 乃初 請 之 邑 。 再 請 之州 。 州伯邑長 。 僉諧 懇願 。 以 大唐 上 元 二年 龍集 辛丑 季秋 月旬 有 九日 。 遂 移 氒 舊 額 。 肇 氒 新居 於 天竺 之山 。 為 真寶地 也 。 天竺 在 故寺 東南 七里 。 名符 佛國 。 山則 我 鄉 。 此蓋 有 開必先 。 陰騭 靈鷲 。 禪師 嘗讀 經典 。 至千 二百五十 人 俱 。 闕 一字 之 而言 曰 。 豈 直多徒 。 亦 堪 集事 。 遂 據 此數 。 以 裒 其人 。 人錢 三 緡 。 共 成法 相 。 行檀 如水 。 品物 如山 。 未盈旬 時 。 我望 充塞 闕 一字 是 邑 也 。 建業 舊 都 。 有 齊梁 遺風 。 乃 聚沙為 塔 。 於 然燈求 記者 。 家 不 無 之 。 易用 受化 。 是 故棟 

梁之才 。 闕 一字里 而 來 。 如陵 如堆 。 班 倕 之伍 。 千里 而 聚 。 如雲如雨 。 乩 之 以 日 。 懋 之 以功 。 栞 山 陊 崖 。 敜 阱 夷 坎 。 芟 蒙蘢 為 顯地 。 剗 崱 闕 一 字 為 康衢 。 其平 如磨 。 其細 如 砥 。 禪師 以心 居中 度殿 。 以背 居後度宇 。 以首 居 高度 臺 。 以足 居下度 室 。 以臂 居北度 廊 。 以手 居南度 門戶 。 授闕 三字 皆 約 我身 。 規圓 之 。 矩方 之 。 縱度 之 。 橫闕 一字 之 。 上協 於 天 。 下協 於 地 。 明協 於 人 。 幽協 於 神 。 然後斯 之 以 斧斤 。 督之以 繩墨 。 審之以 面勢 。 較 之 以方 隅 。 使人 無所惑 也 。 闕 一字 之 以 闕 三字 之 以 榱 桷 。 闕 一字 之 以軒 牖 。 墄 之 以 堦 墀 。 使人 知大壯 也 。 先之以 粉繪 。 後 之 以丹 雘 。 雕之以 金 璧 。 鏤之以 闕二字 使 人觀 巨麗 也 。 春爍 瀛海 。 花敷蓬 萊 。 如鵬斯飛 。 如虹斯 飲 。 色 空相 射 。 晶光 相 廞 。 煌煌 炎炎 。 闕二字 爛爛 。 迴 翔 日月 。 吐納 陰陽 。 弗可得 而 名 也 。 白 蕩 之山 。 以 闕 一字 其 左 。 滄江 之水 。 以 浟 其 右 。 斗辰 之 闕 一字 以 爓 其 上 。 盤龍 之鎮 。 以 抉其 後 。 望 夫南 上以 啟行 。 慈姥東 向 而 奔走 。 勝勢 交闕 三字 放紛 。 蓄靈孕奇 。 闕 一字嵐 泄霧 。 彩章 屢變 。 濃淡 更鮮 。 又 弗 可 得 而 名 也 。 多羅之樹 。 鬱 以 青蔥 。 功德 之水 。 湛而 清淨 。 涌塔浮 於 倒景 。 香剎 彗 於 行 雲 。 貴色 有 薝 蔔 之 花 。 龢 鳴有 迦陵 之 鳥 。 大雄據 師子 之座 。 賁然 當陽 。 太 

子垂瓔 珞 之衣 。 嫣然 列侍 。 相好 闕二字 變化 無窮 。 罔 得 智而知 。 罔 得 億而測 。 巍巍 光大 。 不可 稱量 。 四天 赫臨 。 八部 周護 。 持 殳 秉 矟 。 一何 稜 闕 四字 頤 一何 拗 怒 。 精靈 肸蠁 。 如 在 虛空 。 乎 誘掖 群生 。 虔修 六度 。 撞 鐘鳴 磬 。 以破 昏疑 。 擊 鼓吹 螺 。 以施 號令 。 聞者 開 般若 之智 。 見者 發 菩提 之心 。 或謂 寶生 之國 。 極樂 之土 。 頗 黎為 地 。 黃 金為 繩 。 流泉 浴池 。 珠交霧 幔 。 以此 為念 。 吾 無 靦焉 。 惟禪師 之經 營茲寺 也 。 禎 符景瑞 。 匪朝伊夕 。 五 翬 掊 地 。 神之定 矣 。 三虎 禦寇 。 神之命 矣 。 二木不奪 。 神之正 矣 。 一泉息 濁 。 神之淨 矣 。 靈芝 三秀 。 嬔 谷 嬔 山 。 祥蓮合房 。 于 沼于 沚 。 造門 之 女 。 呴 而 不宿 。 遊方 之 人 。 投而弗禁 。 祅 災起念 而 自 殄 。 危懼 歸心 而 必釋 。 禪師 之道 。 惡可倪 也 。 禪師 之德 。 惡可測 也 。 演 慈悲 之化 。 降淫 慝 之神 。 皆 建廟 立祠 。 血食 不絕 。 近云百祀 。 遠謂 千齡 。 大則 犧牛 之建 。 小則 特豚 之禱 。 以 月 以 日 。 以時 以節 。 弗敢 矯 誣 。 不闕 五字墉 屹然 。 灌木 捎天 。 但 得 而 遵 。 何繇 而 殛 。 禪師 以 為 修道 之 本 。 在 於 利人 。 從 人 之 欲 。 可謂 除 患 。 迺 詣 其 闕 三字視 諦 觀 。 以 日 繫 月 。 身心 靜一 。 或 呈 醜相 。 未 覩 睟 容 。 或 闕 三字 無何 稽首 。 悉 與 受 菩薩淨戒 。 而度 脫 之 。 斬 陰斬陽 。 以為 梵宇 。 取彼 居室 。 寘 之 

金仙 。 寂寥 而 無事 矣 。 城邑 聚落 。 數 百里 閒 。 巫風 遂 消 。 佛道 增長 。 闕 　 字主之教 歟 。 繄 禪師 之力 歟 。 二 相交 修 。 一體 互用 。 不然 者 。 何以 元通 妙感 而 若 是 乎 。 予耳 聆嘉聲 。 目覽懿跡 。 迺 知 闕 一字 聖道者 誰 能慮 始 。 刱 僧 坊 者 闕 四 字 王公大人 。 豪富 長 者 。 國當 全盛 。 家有 貨財 。 然以 更 之 存亡 。 積之年紀 。 人則 盡瘁 。 事 或 藂 殘 。 未有 闕二字乎 禪門 。 獨行 世界 。 時 遭多難 。 道 弗 屢空 。 闕二字 而 闕 一字福 自開 。 晏 坐 而 萬緣斯 湊 。 不 七八 稔 。 指顧 皆 成 。 輪才乎 而 。 奐 才 乎 而 。 若夫 經始 之 善人 。 屋宇 之 闕 一字數 。 紀於碑 陰 之上 。 庶千劫炳 然而 可闕 一字 其辭 曰 。 觀空 匪易 。 闕 一字 相良 難 。 既觀 又 闕 一字揚 波導瀾 。 淵淵 我師 。 體闕 一字 行端 。 經之 精舍 。 粲 以林巒 。 其宇 伊何 。 維 𣒰 與 檀 。 其飾 伊何 。 維琅 與 玕 。 燿燿 華影 。 峨峨 鬱 盤 。 閟 陰迪陽 。 從暑 閱寒 。 世界 非廣 。 渤 澥 非 寬 。 景 於 是 萃 。 思 於 是 殫 。 甘露 之門 。 淨口 以 飡 。 般若 之 闕 二字心 以安 。 右 浹 大江 。 左馳長 干 。 靄 如山青 。 赩 若 霞 丹 。 硉 矹 豐 碑 。 闕二字 不刊 。 追琢 斯文 。 億載 是 觀 。
